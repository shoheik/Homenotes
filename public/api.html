API

Get indices from search query

URL

http://knowhow3.com/search

Method

GET

Parameters

Key Description
query search query. URI enconding needs to be done on your end
know 0 or 1. When you want to search in know, then use 1
how 0 or 1. When you want to search in how, then use 1
exmpale 0 or 1. When you want to search in example, then use 1

Example

$ curl http://knowhow3.com/search -X GET -d "query=linux" -d "know=1"
